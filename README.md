# Vanilla First

**Menos camadas, mais clareza. O movimento Vanilla First é para você entender cada linha e recuperar o controle absoluto sobre o que constrói.**

O desenvolvimento de software atual acostumou os programadores a importar soluções antes mesmo de entender o problema. Criamos uma dependência de frameworks e pacotes externos que resulta em sistemas pesados e muitas vezes vulneráveis. Quando uma ferramenta de terceiros quebra, a aplicação inteira para.

O Vanilla First propõe uma mudança prática. A regra de ouro é: tente o código puro antes de buscar um framework.

Nós valorizamos a exploração máxima do poder nativo de linguagens. Escrever código sem dependências desnecessárias devolve a você a autonomia sobre a arquitetura. Você passa a saber exatamente o que acontece em cada requisição.
